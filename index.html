Curso de Git & Github

Git y Github

Fecha: 01/07/2020

Puedes verlo en: https://platzi.com/clases/git-github

En el curso de Git y Github aprendí a utilizar primero la consola (emulador unix) con ubuntu, aprendi con un curso muy amigable con un profesor entusiasta a moverte dentro de la consola con los principales comandos, y hasa visitar paginas web desde la consola; viendo paginas de solo texto. despues inicie git. Git se maneja totalmente en la consola aprendia a iniciar un "repo", agragar archivos al repositorio haciendo commits, crear ramas y hacer merge, y el siguiemte paso es hacer tu cuenta de github y conectarla con tu repositorio local obviamnete todo por consola, aprendí a hacer pull y push para cargar los archivos que se modificaron, para poder asi compartir tus proyectos en la red social mas grande de developers en el mundo.
